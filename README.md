# fullstack_developer_capstone

Capstone project for the Full-Stack Developer program. This repository hosts the Django backend and React frontend for the Best Cars Dealership experience, including dealer listings, reviews, authentication, and deployment artifacts.

## Repository
- Name: xrwvm-fullstack_developer_capstone
- Owner: Epetaway

## Project
- Title: Best Cars Dealership Portal
- Stack: Django, React, Bootstrap
- Key features: dealer directory, state filtering, dealer detail with reviews, sentiment analysis, authentication (login/register), review submission, and deployed landing pages.
