# Best Cars Dealership - Fullstack Developer Capstone

A full-stack web application for car dealership reviews and management.

## Project Overview

This application provides a platform for users to browse car dealerships, view and submit reviews, and explore vehicle inventory across North America.

## Features

- User authentication (login/register/logout)
- Dealership browsing and filtering
- Review submission and sentiment analysis
- Admin panel for management
- Responsive UI with React frontend
- Django REST API backend
- CI/CD with GitHub Actions

## Repository Structure

- `/server` - Django backend and React frontend
- `/SUBMISSION` - Final submission artifacts and documentation

## Setup Instructions

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the development server

See individual component documentation for detailed setup instructions.

## Technologies Used

- Frontend: React, Bootstrap
- Backend: Django, Django REST Framework
- Database: SQLite
- Deployment: Cloud platform (details in SUBMISSION folder)
- CI/CD: GitHub Actions
